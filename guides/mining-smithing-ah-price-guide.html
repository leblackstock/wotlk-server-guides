Read first: Use Target Buyout as the normal price. Post small stacks. Use Quick for fast sales and High only when supply is scarce. AH durations are 4× normal: 12h ≈ 2 days, 48h ≈ 8 days.
Server cooldown note: Titansteel/Arcanite-style cooldowns are shorter or nonexistent here, so price from input cost + profit margin, not cooldown scarcity alone.

Strategy notes

Best movers

  • Titansteel Bar premium
    Check Titanium Bar + Eternal Fire/Earth/Shadow cost before posting.
  • Saronite Bar / Ore volume
    Strong Wrath BS/Engineering/Jewelcrafting demand.
  • Mithril / Thorium / Truesilver
    Old-world profession catch-up bottlenecks.
  • Enchanting rod blanks
    Single-item convenience sales can beat raw bar pricing.

Check before posting

  • Do not smelt every ore. JC buyers may pay more for prospecting ore.
  • Do not price Titansteel/Arcanite only from cooldown history on this server.
  • Use the separate ore and bar sections to compare value before smelting.
  • For old-world metals, 5-stacks and 20-stacks both matter.

Small reminders

Ore vs bar rule

If Jewelcrafters are buying ore for prospecting, raw ore may beat bars. If Blacksmiths/Engineers are leveling, bars may beat ore.

Node value rule

Node values are route-planning estimates, not guaranteed drops. Rare nodes like Titanium and Khorium are worth stopping for even when the average looks conservative.
